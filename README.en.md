# 基于SSM的在线家教管理系统

#### Description
基于SSM的在线家教管理系统1拥有三种角色

- 管理员：学生管理、老师管理、发布管理、留言管理、回复管理、订单管理等
- 教师：我的订单、我的关注、查看回复留言、登录注册等
- 学生：发布需求、留言、账单信息、关注老师、订单信息、个人中心、聘用教师、登录注册等
